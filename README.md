# Stock Broker Dashboard

A real-time stock broker client dashboard allowing users to subscribe to stocks and view live price updates.

## Project Structure
- `stock_broker_backend`: Node.js + Socket.io server
- `stock_broker`: React + Vite frontend

## How to Run

### Backend
1. Navigate to the backend directory:
   ```bash
   cd stock_broker_backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
   *Server runs on port 8000*

### Client (Frontend)
1. Open a new terminal and navigate to the frontend directory:
   ```bash
   cd stock_broker
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   *Frontend runs on http://localhost:5173*
